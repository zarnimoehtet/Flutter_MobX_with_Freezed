# Flutter MobX with Freezed and Retrofit

### အကြောင်းအရာ
1. လက်ရှိရေးပြီးသမျှ ကို **main branch** တွင် push ထားပါသည်။
2. နောက်ရေးမယ့် အပိုင်းတွေကို **branch** ခွဲပြီး push သွားပါမယ်။
3. လက်ရှိမှာ **State Management** ပြီးထားပါတယ်။
